\subsection{Multi-epoch combination and variability flagging}
\todo{Describe the per-epoch $\to$ per-star step (future
\code{ingest/multi\_epoch.py} + per-dataset handlers in
\code{ingest/combiners/}).
Inverse-variance combine with per-instrument zero-point offsets
applied before the weighted mean and an instrument systematic error
floor (e.g.\ \SI{1.1}{\kms} for DEIMOS) added in quadrature after:
\[
\bar{v} = \frac{\sum_i v_i / \sigma_i^2}{\sum_i 1 / \sigma_i^2},
\qquad
\sigma_{\bar{v}} = \left(\sum_i 1/\sigma_i^2\right)^{-1/2}
\oplus \sigma_\text{sys}.
\]
Variability flag via $\chi^2 = \sum_i (v_i - \bar{v})^2 / \sigma_i^2$
against a $p$-value threshold (e.g.\ $p < 0.01$); single-epoch stars
remain unflagged.}
