Posterior sampling was performed with the nested-sampling package
\textsc{dynesty}~\citep{speagle2020,koposov2024dynesty}, which implements
the nested-sampling algorithm of \citet{skilling2004,skilling2006}.
We used multi-ellipsoidal bounds~\citep{feroz2009multinest} with the
random-walk evolution scheme~\citep{skilling2006} (\texttt{bound="multi"},
\texttt{sample="rwalk"}).
